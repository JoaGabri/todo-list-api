# API de Todo Task com Flask e SQL Alchemy / Todo Task API with Flask and SQL Alchemy
Esta é uma API de Todo Task desenvolvida utilizando Flask, um framework web em Python, e SQL Alchemy, uma biblioteca de mapeamento objeto-relacional.
Foi meu primeiro contato com APIS Em Python então decidi que fosse algo mais simples de se fazer, apenas para conhecer as principais ferramentas.

This is a Todo Task API developed using Flask, a Python web framework, and SQL Alchemy, an object-relational mapping library.
It was my first contact with APIs in Python, so I decided to make it simple to get acquainted with the main tools.

#Funcionalidades / Features
A API possui as seguintes funcionalidades:
Criar uma nova tarefa.
Marcar uma tarefa como concluída ou não concluída.
Excluir uma tarefa.

The API has the following features:

Create a new task.
Mark a task as completed or not completed.
Delete a task.


#Requisitos de Instalação / Installation Requirements
Antes de executar o projeto, é necessário ter os seguintes requisitos instalados:

Before running the project, you need to have the following requirements installed:
```
pip install Flask Flask-SQLAlchemy SQLAlchemy
```

Executando a API / Running the API


Para executar a API, utilize o seguinte comando:

To run the API, use the following command:


```
Flask run
```
Apos isso é só acessar o LocalHost.

After that, you can access the localhost.
